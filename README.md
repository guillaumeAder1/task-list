## A visual representation for deliverables in a matrix



[Live demo](https://guillaumeader1.github.io/task-list/public/)

## Install
Install dependencies via npm
```
    npm install
```
Verified environment
```
$ node --version
    v6.11.0
$ npm --version
    5.5.1
```

## Run
Run locally
```
    npm run start
```
